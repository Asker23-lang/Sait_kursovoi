Импорт товаров

Загрузите товары из CSV файла. Существующие товары будут обновлены

Формат CSV файла

CSV файл должен содержать следующие столбцы (в указанном порядке):

name,description,price,image,sizes,tags,stock_quantity,in_stock

Примеры:

Строка с одним размером:
"Футболка Basic","Белая футболка из хлопка","2990","","[""M""]","[""Новинка""]","50","1"
Строка с несколькими размерами:
"Джинсы Slim","Темно-синие джинсы","15990","jeans.jpg","[""S"",""M"",""L"",""XL""]","[""Хит""]","25","1"

Важно:

  • Размеры и теги должны быть в формате JSON массива
  • Цена указывается в тенге без пробелов
  • Изображения должны быть предварительно загружены в папку uploads
  • Пустые значения можно оставлять пустыми (но не удалять столбцы)
